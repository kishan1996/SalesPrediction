## 🚀 Sales Prediction
```
Create an End to End Machine Learning Project which predict the sales of stores according to DateTime dataset given.
```

### 🔥 Technologies Used
``` 
1. Python 
2. Flask
3. HTML,CSS
4. Moduler Coding 
```

## 👷 Initial Setup 

```commandline
conda create --prefix ./env python=3.9
conda activate ./env 
pip install -e .
pip install -r requirements.txt 
```

## 📉 Train the model

```commandline
cd src
python main.py
```

## 🔥 Running the project 
```commandline
python main.py
```
